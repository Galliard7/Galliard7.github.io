## Overview

A remote control interface for Claude Code sessions, operated through Telegram. CC-Remote enables sending commands, monitoring session activity, and managing long-running tasks from a mobile device.

## Features

- Remote command execution and session monitoring
- Handoff protocol for transferring context between sessions
- Process management (start, monitor, kill)
- Telegram-native UX with structured message formatting
