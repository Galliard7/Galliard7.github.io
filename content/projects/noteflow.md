## Overview

A task and idea management system designed for developer workflows. NoteFlow provides a lightweight board for capturing tasks, reminders, and ideas, with bidirectional integration into a project management layer (Mission Control).

## Features

- Task, reminder, and idea item types with flexible metadata
- Subnotes for incremental progress tracking
- Project tagging and cross-referencing
- Bidirectional sync with Mission Control cards
- CLI and API interfaces for automation
